Добавление изменений:
git add . - добавляет все изменения
git add index.html - добавляется только файл index.html
git commit -m "text commit" - делается комит с комментарием
git push origin master - пушит все коммиты и их можно увидеть на github

git pull - обновление локального файла до последней версии гита
git branch - показывает все ветки
